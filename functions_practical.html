Micro-learning for Java Code - Functions

Functions

Write a program that defines a function 'add' which takes two integers as parameters and returns their sum. Call the function with the values 10 and 5 and print the result.
Create a program that defines a function 'greet' which takes a String parameter 'name' and prints a greeting message. Call the function with the value 'Alice'.
Write a program that defines a function 'square' which takes a double parameter and returns its square. Call the function with the value 4.5 and print the result.
Create a program that defines a function 'max' which takes three integers as parameters and returns the maximum value among them. Call the function with the values 10, 20, and 15 and print the result.
Write a program that defines a function 'isEven' which takes an integer parameter and returns true if the number is even, false otherwise. Call the function with the value 8 and print the result.
Create a program that demonstrates method overloading by defining two functions named 'multiply': one that takes two integers and another that takes three integers. Print the results of both.

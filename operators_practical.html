Micro-learning for Java Code - Operators

Operators

Write a program that declares two integer variables, 'a' and 'b', and assigns them values of 10 and 20, respectively. Calculate and print the sum of 'a' and 'b'.
Create a program that declares two double variables, 'x' and 'y', and assigns them values of 5.5 and 2.5, respectively. Calculate and print the result of 'x' divided by 'y'.
Write a program that uses relational operators to compare two integers, 'num1' and 'num2'. Assign them values of 15 and 10, respectively. Print 'num1 is greater' if 'num1' is greater than 'num2', otherwise print 'num1 is not greater'.
Create a program that declares a boolean variable 'isSunny' and assigns it a value of true. Use a logical operator to check if 'isSunny' is true and print 'It's a sunny day!' if it is true.
Write a program that declares an integer variable 'score' and assigns it a value of 85. Use the ternary operator to print 'Pass' if 'score' is greater than or equal to 50, and 'Fail' otherwise.

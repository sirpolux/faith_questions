Micro-learning for Java Code - Conditional Statements

Conditional Statements

Write a program that checks if a number is positive, negative, or zero. Declare an integer variable 'number' and assign it a value of 10. Print whether the number is positive, negative, or zero.
Create a program that determines if a given year is a leap year. Declare an integer variable 'year' and assign it a value of 2024. Print 'Leap Year' if it is a leap year, otherwise print 'Not a Leap Year'.
Write a program that takes a grade (integer) and prints 'Pass' if the grade is 50 or above, and 'Fail' otherwise. Declare an integer variable 'grade' and assign it a value of 65.
Create a program that checks the value of an integer variable 'score' and prints 'Excellent' if the score is above 90, 'Good' if the score is between 75 and 90, and 'Needs Improvement' otherwise. Declare 'score' and assign it a value of 85.
Write a program that checks if a character (char) is a vowel or a consonant. Declare a char variable 'letter' and assign it a value of 'A'. Print 'Vowel' if it is a vowel, otherwise print 'Consonant'.
